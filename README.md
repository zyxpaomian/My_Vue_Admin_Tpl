# My_Vue_Admin_Tpl

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli)

``` bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 编译安装
npm run build
```

# 文件
* build - webpack config files
* config - webpack config files
* dist - build
* src -应用源码
    * assets - 静态资源，存放图片,icon之类的
    * components - 基础页面组件，如导航栏等等
    * router - 路由
    * store - vuex,以及部分共享后的数据使用localstorage
    * style - 基本样式，主题的颜色，默认是天蓝色
    * utils - 工具类封装，如权限和钩子函数的封装
    * views - 视图层，所有页面在里面
    * App.vue
    * main.js - main file
* static - static assets
* .babelrc - webpack 构建工具
*
